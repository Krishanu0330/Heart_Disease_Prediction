The dataset used in this project contains several medical attributes such as age, sex, chest pain type, blood pressure, cholesterol level, fasting blood sugar, resting ECG results, maximum heart rate, exercise-induced angina, and other clinical measurements. These features are used to train machine learning algorithms to classify whether a patient is likely to have heart disease.

The workflow of the project includes data preprocessing, exploratory data analysis, feature selection, model training, and evaluation. Various machine learning algorithms such as Logistic Regression, Decision Tree, Random Forest, and Support Vector Machine are implemented to compare performance and accuracy.

After training the models, evaluation metrics such as accuracy, precision, recall, and confusion matrix are used to measure the effectiveness of the prediction system. The best-performing model is selected to make predictions on new patient data.

This project demonstrates how machine learning can be applied in the healthcare domain to assist in disease prediction and support medical decision-making.
